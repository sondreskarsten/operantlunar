# Plot a discounting fit

Plot a discounting fit

## Usage

``` r
plot_discounting(delays, indiff, fit)
```

## Arguments

- delays:

  Delays.

- indiff:

  Indifference amounts.

- fit:

  Output of
  [`fit_discounting()`](https://sondreskarsten.github.io/operantlunar/reference/fit_discounting.md).

## Value

A ggplot object.
