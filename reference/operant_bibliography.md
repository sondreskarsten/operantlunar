# Operant-conditioning bibliography

The bundled handbook corpus (held in GCS) plus the primary sources cited
in
[`operant_glossary()`](https://sondreskarsten.github.io/operantlunar/reference/operant_glossary.md).
Handbook entries include their GCS location.

## Usage

``` r
operant_bibliography()
```

## Value

A tibble with columns `key`, `type`, `citation`, `gcs_uri`.

## Examples

``` r
operant_bibliography()
```
