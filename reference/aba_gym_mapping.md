# ABA-to-gym mapping

How each toolkit instrument reframes a reinforcement-learning problem:
the ABA theory as an alternative analytic framework for gym
environments.

## Usage

``` r
aba_gym_mapping()
```

## Value

A tibble with `instrument`, `rl_problem`, `gym_mechanism`,
`diagnoses_or_changes`.

## Examples

``` r
aba_gym_mapping()
```
