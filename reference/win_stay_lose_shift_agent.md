# Win-stay-lose-shift agent (operant heuristic)

Win-stay-lose-shift agent (operant heuristic)

## Usage

``` r
win_stay_lose_shift_agent(n_actions = 2L, p_explore = 0)
```

## Arguments

- n_actions:

  Number of actions.

- p_explore:

  Probability of a random action regardless of history.

## Value

A list of agent callables.

## Examples

``` r
names(win_stay_lose_shift_agent())
```
