# Basic behavioural phenomena (candidate-instrument pipeline)

Reproduced behavioural phenomena that are not (yet) operationalised and
literature-verified into instruments. A phenomenon graduates into
[`aba_toolkit()`](https://sondreskarsten.github.io/operantlunar/reference/aba_toolkit.md)
once its read-out tracks the mechanism and is verified; a look-alike
curve produced for the wrong reason does not qualify.

## Usage

``` r
basic_phenomena()
```

## Value

A tibble with `phenomenon`, `status`, `env`, `shows`, `note`.

## Examples

``` r
basic_phenomena()
```
