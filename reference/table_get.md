# Get a row from the table, defaulting to zeros

Get a row from the table, defaulting to zeros

## Usage

``` r
table_get(table, s, n_actions)
```

## Arguments

- table:

  A table from
  [`make_table()`](https://sondreskarsten.github.io/operantlunar/reference/make_table.md).

- s:

  Character state key.

- n_actions:

  Number of actions.

## Value

Numeric vector of length `n_actions`.
